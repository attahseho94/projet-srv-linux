# Projet BTS SIO SISR - Serveur Linux

## Objectif
Mise en place d’un serveur Linux Ubuntu avec services Apache et SSH sécurisé.

## Technologies utilisées
- Ubuntu Server
- Apache2
- SSH
- Firewall UFW
- VirtualBox

## Fonctionnalités
- Serveur web Apache fonctionnel
- Accès SSH sécurisé par clé
- Firewall activé
- Analyse des logs système
